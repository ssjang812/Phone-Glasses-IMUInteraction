# Phone-Glasses-IMUInteraction: Integrated Interaction Module for Smart Augmented Reality (AR)

---

## 1. Introduction

This project is an integrated interaction module developed to efficiently receive and process data from **Microsoft HoloLens 2** and **Xsens DOT IMU sensors** within a smartphone application environment.

The primary objective is to establish an efficient UI/UX platform in the AR environment by implementing remote **input event reception** via a smartphone and providing **3D object manipulation** functionalities based on the received data.

---

## 2. Development Environment

* **Unity Engine Version:** 2022.3.60f1
* **Primary Target Device:** Microsoft HoloLens 2
* **Sensor Integration:** Xsens DOT IMU Sensor

---

## 3. Core Features

| Feature Category | Description |
| :--- | :--- |
| **HoloLens 2 Integration** | HoloLens 2 **Input Event Reception and Processing Module** via smartphone application. |
| **Object Manipulation** | **3D Object Control and Manipulation** functionality within the HoloLens 2 environment, based on received input events. |
| **IMU Sensor Data** | Real-time **Orientation and Motion Data Reception Module** from the Xsens DOT IMU sensor. |

*Detailed usage instructions and module integration guides will be updated later.*
